# Project Portfolio Analysis Dataset

## Overview

This is a comprehensive project portfolio dataset containing 250 projects with 20 detailed attributes each. The dataset is designed for deterministic portfolio prioritization analysis following industry standards and best practices.

## Dataset Characteristics

- **Total Projects**: 250
- **Attributes per Project**: 20
- **Budget Range**: $72,431 - $4,959,384
- **Duration Range**: 3 - 35 months
- **Expected ROI Range**: 7.05% - 298.9%
- **Annual Benefit Range**: $22,289 - $1,995,342

## Project Categories

The dataset includes projects across 10 strategic categories:
- Infrastructure
- Technology
- Process Improvement
- Product Development
- Cost Reduction
- Revenue Growth
- Risk Mitigation
- Compliance
- Sustainability
- Innovation

## Attributes

| Attribute | Description | Type | Range |
|-----------|-------------|------|-------|
| Project_ID | Unique project identifier | String | PRJ-0001 to PRJ-0250 |
| Project_Name | Descriptive project name | String | - |
| Category | Strategic project category | String | See categories above |
| Budget_USD | Total project budget | Integer | $72,431 - $4,959,384 |
| Duration_Months | Expected project duration | Integer | 3 - 35 months |
| Expected_ROI_Percent | Expected return on investment | Float | 7.05% - 298.9% |
| Risk_Level | Project risk classification | String | Very Low, Low, Medium, High, Very High |
| Strategic_Alignment | Alignment with strategy | String | Critical, High, Medium, Low |
| Resource_Availability_Percent | Available resources | Integer | 20% - 100% |
| Implementation_Complexity_Score | Complexity rating | Integer | 1 - 10 |
| Stakeholder_Support_Score | Stakeholder support level | Integer | 1 - 10 |
| Timeline_Feasibility_Score | Timeline feasibility | Integer | 1 - 10 |
| Dependency_Count | Number of project dependencies | Integer | 0 - 5 |
| Annual_Benefit_USD | Annual monetary benefit | Integer | $22,289 - $1,995,342 |
| Critical_Path_Days | Critical path duration | Integer | 30 - 1000 days |
| Team_Availability_Percent | Team availability | Integer | 30% - 100% |
| Regulatory_Requirement | Regulatory compliance required | Boolean | True/False |
| Customer_Impact_Score | Customer impact rating | Integer | 1 - 10 |
| Competitive_Advantage_Score | Competitive advantage rating | Integer | 1 - 10 |
| Operational_Efficiency_Gain_Percent | Operational efficiency gain | Float | 0% - 50% |
| Quality_Improvement_Score | Quality improvement rating | Integer | 1 - 10 |

## File Format

The dataset is provided in two formats:
- **CSV**: `project_portfolio_dataset.csv` (comma-separated values)
- **JSON**: `project_portfolio_dataset.json` (JSON Lines format)

## Usage

This dataset is intended for:
